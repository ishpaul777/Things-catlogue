# OOPS catlog

This is program which can take care of organising your things and information related to them. One can 
> - Add new books
> - Add new labels
> - Add new games
> - Add new authors
> - Add new music albums.
> - Add new genre
> - And see the list of all items will be acheivable in a beautiful and well-organized way with this project!


## Built With

- Ruby

## Getting Started

- If you dont have Ruby installed on your computer, you can download it from here.
- Clone the project in your machine
- To Start the programs

```ruby main.rb```
